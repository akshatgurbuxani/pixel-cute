# Pixel Hugs ♡

A cute pixel-art meadow for Baby — animated pink gradients, bouncing cartoon sprites, tap-to-spawn cuties.

## Run

```bash
npm install
npm run dev
```

## Deploy

```bash
npx vercel
```

## Customize

Edit `src/data/sprites.ts`:
- `HER_NAME`
- `MESSAGES` — speech bubble text
- `SPRITES` — pixel art grids (each letter = a color from `PALETTE`)

## Stack

React + Vite + Framer Motion + canvas-confetti + hand-drawn pixel SVG sprites
